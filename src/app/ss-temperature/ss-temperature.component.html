2014 Global (Land) Temperature Contour
Monthly-mean air temperature raw data for the year 2014 comes from this source. The raw data interpolates to a 0.5 by 0.5 degree grid resolution (centered on 0.25 degree). However, I use R to reformat the data in to a JSON file interpolated to a 2 by 2 degree grid resolution for this visualization.
